A modern frontend restaurant website built to practice and showcase real-world frontend concepts like component-based UI, routing, reusable components, and clean layout structure.

This project includes:
- Home, Menu, Services, About, and Contact pages
- Reusable UI components (cards, navbar, footer)
- Clean and modern UI/UX
- Frontend-only architecture

## Tech Stack
- React
- React Router
- Tailwind CSS

## About This Project
This project was created as a learning and practice project.  
The initial structure and features were generated using **Lovable AI**, and then reviewed, organized, and customized by me.

The goal was not just design, but understanding how a real frontend project is structured and managed.
